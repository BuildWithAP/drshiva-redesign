# Dr. Shiva Harikrishnan — Redesign Mockup

Direction: **Clinical Confident** (editorial)

## Design system
- Type: Fraunces (display, optical sizing) + Geist (sans body)
- Palette: Ink `#0E2435`, Bone `#F6F1E8`, Peacock teal `#2F6B6B`, Ochre `#DCC9B0`
- Style: editorial-clinical, asymmetric hero, generous whitespace, no carousels

## Files
- `index.html` — Home (hero, marquee, stats, about teaser, services, quote, gallery, CTA)
- `about.html` — Bio, education timeline, certifications, philosophy
- `services.html` — Six services itemised with bullets and imagery
- `contact.html` — Booking form, clinic info, map placeholder
- `styles.css` — Shared design system
- `script.js` — Scroll-reveal observer

## Photos
All photography is hotlinked from `drshivahk.com` (existing CDN). Replace with hosted versions before going live.

## Deploy
Static — drop the folder into any web host (GitHub Pages, Netlify, Vercel, S3) or push directly.

```bash
cd drshiva-redesign
git init
git add .
git commit -m "Clinical Confident redesign mockup"
```
